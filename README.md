# Laptop Issue Management System | ServiceNow

## Project Overview
Developed a custom ServiceNow application to manage laptop service requests. The project was built using a ServiceNow Personal Developer Instance (PDI) to gain hands-on experience with ServiceNow administration and development concepts.

## Features Implemented
- Created a custom ServiceNow application.
- Designed a custom table and form for laptop issue requests.
- Configured reference, choice, date/time, currency, attachment, and auto-number fields.
- Created and managed users, groups, and roles.
- Imported data using Import Sets and Transform Maps.
- Migrated configurations using Update Sets.
